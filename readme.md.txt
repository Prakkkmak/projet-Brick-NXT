=====	Accéléro-stabilisateurr v10b - Guide d'utilisation	=====

	
	Objectif du projet :
	
	Le module à pour but de stabiliser en temps réel une partie de la brique à l'horizontale,
	il intègre ainsi un bouton permettant de désactiver ou non la stabilisation en permanence,
	ainsi que deux moteurs pouvant être actionnésvde façon à dérégler la position du bras mécanique,
	et tester la stabilisation automatique.
	
	Le bouton poussoir se situant à gauche de la brique permet de désactiver la stabilisation automatique,
	le mode est alors affiché sur l'écran de la brique : "System ON" lorsque la stabilisation est activé,
	"System OFF" lorsque la stablisation est désactivée.
	
	Les deux flèches se situant sur la brique NXT permettent respectivement de bouger en arrière et en avant
	le bras mécanique afin de "déstabiliser" la structure, et ainsi tester la stabilisation automatique intégrée.
	
	
	Pour lancer le programme :
	
	Aller dans My Files -> Software Files et lancer le programme "AccStab".
	
	Pour quitter le programme :
	
	Appuyer sur le bouton retour de la brique.
	
	
	Guide d'utilisation :
	
	- BOUTON GAUCHE : reculer le bras mécanique
	- BOUTON DROITE : avancer le bras mécanique
	- BOUTON POUSSOIR : Désactiver/Activer le système de stabilisation en temps réel
	- BOUTON RETOUR : Quiter le programme
	
	Pour modifier l'angle approximatif de stabilisation, il faut changer la variable "stabilisationDegres" dans le programme.
